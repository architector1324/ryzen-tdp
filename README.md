# Ryzen TDP
Simple Ryzen TDP manager shell scripts using [RyzenAdj](https://github.com/FlyGoat/RyzenAdj).

## Usage
Use it manually with shell scripts:
1. Set TDP
```bash
# set tdp to 8 watt
./set-tdp.sh 8
```

```bash
./tdp-gui.sh
```

2. Get TDP + Limits:
```bash
./get-tdp.sh
```

```bash
./get-tdp-lim.sh
```
